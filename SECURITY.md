# Securitybeleid

Commit nooit wachtwoorden, API-geheimen, service-role keys, reserveringsback-ups, persoonsgegevens, OCR-bronbestanden of lokale bestandspaden.

De GitHub Actions-workflow scant het Pages-artifact op bekende gevoelige patronen en weigert publicatie wanneer zulke gegevens worden gevonden.

Meld beveiligingsproblemen privé via `info@dvdstegeef.be`; plaats geen persoonsgegevens in een publieke GitHub issue.
